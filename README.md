🎨 Mood Art Generator
A generative art tool that creates unique abstract pieces based on your mood. No two generations are the same — every click produces a one-of-a-kind artwork powered by a seeded random algorithm and SVG rendering.
6 moods, 6 algorithms:

Calm → soft overlapping blobs
Joyful → radiating sunbursts
Melancholy → drifting wave lines
Electric → sharp geometric grids
Wild → chaotic shapes and polygons
Dreamy → floating ellipses and curves

Built with: Vanilla JavaScript · SVG · CSS · Zero dependencies · No frameworks · No build step
Features: Export any piece as a high-res PNG · Fully responsive · Dark-themed UI · Works offline
